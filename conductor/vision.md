# Vision

**Summary**: A portable agent harness that can be dropped into any coding project as a powerful, structured AI accessory.

---

## What We're Building

A self-contained system of skills, agents, and behavioral contracts that makes Claude Code disciplined, context-aware, and orchestration-capable — regardless of which project it's working in.

The harness travels with you. Drop it into a TypeScript/Next.js app, a Koa API, a Flutter mobile project, or a Terraform infrastructure repo — and the same structured workflow, agent library, and knowledge base are immediately available.

## Why

Without structure, AI agents are eager and undisciplined — they skip requirements, skip tests, assume intent, and accumulate context debt. This harness enforces the workflow gates that prevent those failure modes, so every session starts from a known-good state.

## Success Looks Like

- Any new coding project can be bootstrapped with this harness in under 5 minutes
- Every agent session starts with project context loaded, not guessed
- New features go through brainstorm → spec → plan → implement → review without manual reminders
- The knowledge base grows with every session and is queryable by Claude directly
